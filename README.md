# ⌨️ Typing Speed Battle

A fast and interactive typing game where users race against time to test their typing speed and accuracy.

## 🚀 Features

- ⏱️ Real-time typing race with countdown timer  
- 🎯 Live character validation (correct/incorrect highlight)  
- 📊 WPM (Words Per Minute) calculation  
- ✅ Accuracy tracking  
- 🏆 Local leaderboard (stored using localStorage)  
- 🔁 Restart / Try Again functionality  
- 🌙 Clean UI with responsive design  

## 🧠 How It Works

- A random paragraph is displayed  
- Timer starts when user begins typing  
- Each keystroke is validated instantly  
- Game ends when:
  - Timer runs out OR  
  - Paragraph is completed
